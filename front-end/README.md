# Restaurant Map

一個基於 Vue 3 和 Spring Boot 的餐廳地圖應用，讓用戶可以探索、分享和評論餐廳。

## 技術棧
### 前端
- Vue 3
- Element Plus
- Axios
- Vue Router
- Pinia
- Vite

### 後端
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- JWT

## 專案結構
```
restaurant-map/
├── front-end/          # Vue 3 前端專案
└── back-end/           # Spring Boot 後端專案
```

## 快速開始

### 前端
```bash
cd front-end
npm install
npm run dev
```

### 後端
```bash
cd back-end
mvn install
mvn spring-boot:run
```
